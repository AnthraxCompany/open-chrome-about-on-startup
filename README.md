# Chrome起動時に、chrome://settings/help ページを追加で開くことで 更新チェックを行わせる拡張

# Chrome への読み込みと動作確認手順

1 Chrome で chrome://extensions/ を開く。

2 右上の「デベロッパーモード」をオンにする。

3 「パッケージ化されていない拡張機能を読み込む」をクリックし、
ダウンロードしたフォルダ（open-chrome-about-on-startup/）を選択。

4 拡張が一覧に表示され、エラーが出ていないことを確認。

5 Chrome を完全に終了させてから再起動する。
すべてのウィンドウを閉じるか、chrome://restart をアドレスバーに入力して実行。

6 再起動後、
いつも通りの起動タブ（前回のセッション復元やスタートページなど）に加えて、
chrome://settings/help が新しいタブとして自動で開いている
ことを確認。
